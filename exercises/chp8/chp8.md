### 문제 2. “관계 데이터 모델의 기본이 되는 릴레이션을 구성하는 속성들의 집합”에 적합한 용어는?
정답: 릴레이션 스키마

### 문제 3. 회원이 어떤 고등학교를 졸업했는지를 표현한 ER 다이어그램을 릴레이션 스키마로 변환한 다음, 관계 릴레이션을 단순화하여 표현하시오.
정답:

```text
회원(회원번호, 이름, 휴대폰)
PK: 회원번호

고등학교(학교번호, 학교명, 주소)
PK: 학교번호

졸업하다(회원번호, 학교번호, 입학일자, 졸업일자)
PK: 회원번호, 학교번호
FK: 회원번호 → 회원(회원번호)
FK: 학교번호 → 고등학교(학교번호)
```

### 문제 4. 지역별로 회원이 어떤 카페에 가입했는지를 표현한 ER 다이어그램을 릴레이션 스키마로 변환하시오.
정답:

```text
회원(지역번호, 가입순서, 이름, 휴대폰)
PK: 지역번호, 가입순서

카페(id, 이름, 개설자, url)
PK: id

가입하다(지역번호, 가입순서, id, 가입일자, 역할)
PK: 지역번호, 가입순서, id
FK: 지역번호, 가입순서 → 회원(지역번호, 가입순서)
FK: id → 카페(id)
```

### 문제 5. 학생이 어떤 외부 활동에 참여했는지를 표현한 ER 다이어그램을 릴레이션 스키마로 변환하시오.
정답:

```text
학생(학번, 이름, 전공)
PK: 학번

외부활동(학번, 번호, 기관명, 활동내역, 참여기간)
PK: 학번, 번호
FK: 학번 → 학생(학번)
```

### 문제 6. 모든 학생이 반드시 하나의 방과후 활동에 참가해야 하고, 학생 가운데 1명이 각 방과후 활동의 대표를 맡는다는 사실을 표현한 ER 다이어그램을 릴레이션 스키마로 변환하시오.
정답:

```text
학생(번호, 이름, 학년, 활동번호)
PK: 번호
FK: 활동번호 → 방과후활동(번호)

방과후활동(번호, 활동명, 주요활동, 모임장소, 대표학생번호)
PK: 번호
FK: 대표학생번호 → 학생(번호)
```

### 문제 7. 교수가 어떤 연구회에 가입했는지를 표현한 ER 다이어그램을 가장 바람직한 형태의 릴레이션 스키마로 변환하시오.
정답:

```text
연구회(번호, 명칭, 목적)
PK: 번호

교수(교수번호, 이름, 직급, 입사년도, 연구회번호, 가입일)
PK: 교수번호
FK: 연구회번호 → 연구회(번호)
```

### 문제 8. 1명의 학생이 최대 5명의 다른 학생에 대해 멘토링하는 관계를 가장 간단한 형태의 릴레이션 스키마로 변환하시오.
정답:

```text
학생(학번, 이름, 학과, 휴대폰, 멘토학번)
PK: 학번
FK: 멘토학번 → 학생(학번)
```

### 문제 9. 콘서트를 개최할 때 뮤지션과 기획사, 콘서트홀이 연관되는 관계에서 콘서트개최하다 관계를 릴레이션 스키마로 변환하시오.
정답:

```text
콘서트개최하다(id, 회사번호, num, 시작일자, 종료일자)
PK: id, 회사번호, num
FK: id → 뮤지션(id)
FK: 회사번호 → 기획사(회사번호)
FK: num → 콘서트홀(num)
```

### 문제 10. 하나 이상의 자격증을 소지한 응시자 개체를 표현한 ER 다이어그램을 릴레이션 스키마로 변환하시오.
정답:

```text
응시자(접수번호, 이름, 휴대폰, 희망부서, 최종학력)
PK: 접수번호

응시자자격증(접수번호, 자격증)
PK: 접수번호, 자격증
FK: 접수번호 → 응시자(접수번호)
```
